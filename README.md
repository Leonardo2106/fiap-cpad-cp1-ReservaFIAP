### 📋 Sobre o Projeto:

A **ReservaFIAP** é um **app mobile** que possibilita **alunos da FIAP reservarem salas** nos campus com intuito de **criar comunidades**, **grupos de estudos** ou até mesmo para discutir sobre: **Challenge**, **Iniciação Científica** ou **Brainstorms**. O nosso objetivo é **facilitar** e **possibilitar** essa criação de comunidades de uma maneira **organizada** e **inteligente**.

### 📦 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/Leonardo2106/cpad-cp01.git
```

2. Dentro do repositório instale as dependências e inicia a aplicação:
```bash
cd ReservaFIAP

npm install

npm start
```
